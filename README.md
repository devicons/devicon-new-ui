# DeviconNewUi

Prerequisites:
- [Node.js](https://nodejs.org/en/download)

### Development
Install dependencies:
```bash
npm install
```

Build and run app locally:
```bash
ng serve
```

Run unit tests:
```bash
ng test
```

Deploy:
```bash
ng deploy --base-href=https://<username>.github.io/<repo-name>/
```

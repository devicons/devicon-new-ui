# DeviconNewUi

Prerequisites:
- [Node.js](https://nodejs.org/en/download)

### Development
Install dependencies:
```
npm install
```

Build and run app locally:
```
npm run start
```

Run unit tests:
```
npm run test
```

Lint:
```
npm run lint
```

Format:
```
npm run format
```

Deploy to gh-pages branch:
```
ng deploy --base-href=https://<username>.github.io/<repo-name>/
```
